## 1. 프로젝트 소개

<p align="center">
사용자가 날씨 정보를 기반으로 옷차림을 추천받을 수 있는 애플리케이션 <br>
현재 위치의 날씨 데이터를 실시간으로 확인하고, 해당 날씨에 적합한 옷차림을 AI를 통해 제안받을 수 있다. 
</p>


## 2. 주요 기능
